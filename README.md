# AI-Powered Customer Churn Analytics

## Overview
End-to-end churn prediction system with ML, explainability, and AI-generated business insights.

## What I Built
- Churn prediction using Logistic Regression
- Driver analysis using feature importance
- AI explanation layer using LLM (Ollama)
- Power BI dashboard for business decisions

## Key Innovation
Instead of running LLM per customer, I generated explanations at the **driver-pattern level**, making the system scalable and efficient.

## Tools Used
Python, Pandas, Scikit-learn, Power BI, Ollama (LLM)

## Output
- Risk segmentation (High / Medium / Low)
- Top churn drivers
- AI-generated reason + action
- Interactive dashboard

## Business Impact
Helps identify high-risk customers and take targeted retention actions.
